lucille パッケージ構成
======================

::

  doc/                  ドキュメント

  shaders/              サンプルシェーダファイル
  src/                  レンダラコアのソースコード

  tools/                ツールなど
  
``src`` の詳細についてはソースコード構成 :ref:`xref_source` を参照のこと.
