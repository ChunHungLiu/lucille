lucille 概要¶

プロジェクトの目標¶

lucille の目標は, プロダクションレンダラや商用レンダラよりも優る大域照明レンダリング機能とその高速な実行性能をオープンソースで提供することにより, コンピュータグラフィックスコミュニティへ lucille レンダラおよび大域照明レンダラ技術の普及を図ることである.

ストラテジ¶

プロジェクトの目標に基づき、lucille が目指すより具体的な戦略は以下の通りである.

  • 最新大域照明アルゴリズムの提供(VPL, MLT, etc)
  • CPU リソースを余すことのない高速なレンダリング処理(SIMD 処理, マルチスレッド実行, 並列レンダリング, etc)
  • ビームレイトレーシングによるエイリアスフリーのレンダリング画像生成
  • JIT コンパイルされ高速に実行されるシェーダ言語のサポート

lucille プロジェクトのストラテジでは、大域照明アルゴリズムの実装とその高速な実装という二点に重点を置いている.
