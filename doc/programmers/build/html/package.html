lucille パッケージ構成¶

doc/                  ドキュメント
include/              インターフェイスヘッダファイル
lib/                  ビルド後にライブラリが置かれる場所
shaders/              サンプルシェーダファイル
src/                  レンダラコアのソースコード
tests/                テスト関連
tools/                ツールなど

src の詳細についてはソースコード構成 ソースコード構成 を参照のこと.
