Binary release of ShaderEngine
==============================

Syoyo Fujita

Requirements
------------

 - Intel Mac
 - Python

How to execute
--------------

::

  $ ./slengine <shader.sl>


Limitation
----------

This release is a technical preview. Many features are not implemented yet. matte.sl and wood.sl works well, but other shaders might not work.
